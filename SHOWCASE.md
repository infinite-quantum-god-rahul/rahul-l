### 🛠️ **Development Setup**
```bash
# Clone the repository
git clone https://github.com/infinite-quantum-god-rahul/rahul-l.git
cd rahul-l

# Setup backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

# Setup mobile app
cd sml_mobile_app
flutter pub get
flutter run

# Setup frontend
npm install
npm start
```
